Agent admitted failure to sign using the key

作者：史英建 | 发表于28 Apr 2011 标签： git ssh

使用:

1ssh-keygen -t rsa -C "your_email@youremail.com"

产生出 idrsa, idrsa.pub, 后用 ssh git@github.com 连线时却出现下述讯息:

1Agent admitted failure to sign using the key
2Permission denied (publickey).

解法

于自己的机器上, 执行 ssh-add, 会出现下述讯息.

1Identity added: /home/user/.ssh/id_rsa (/home/user/.ssh/id_rsa)

再次连线就正常了~

1PTY allocation request failed on channel 0
2Hi shiyj! You've successfully authenticated, but GitHub does not provide shell access.
3Connection to github.com closed.
