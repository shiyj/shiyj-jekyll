ubuntu 环境配置

作者：史英建 | 发表于10 Oct 2011 标签： ubuntu

ubuntu init

最近重新安装了系统，配置ubuntu很麻烦，把所需的都记录下来，不用每次都看别人的：

1 安装git

1sudo apt-get install git-core git-gui git-doc
2ssh-keygen -t rsa -C "shiyj.cn@gmail.com"

把id_rsa.pub内容加入到github中。

2 c/c++开发环境

1sudo apt-get  install  build-essential

####安装ide Code::Blocks及图形界面wxwidgets

1sudo apt-get install build-essential
2sudo apt-get install codeblocks-contrib
3sudo apt-get install libwxgtk2.8-dev

3 openssl

1sudo apt-get install libssl-dev

4 编译 ruby所需的依赖

1sudo apt-get install zlib1g-dev
2sudo apt-get install libx11-dev

5 Ubuntu GIS源

1deb http://ppa.launchpad.net/ubuntugis/ppa/ubuntu lucid main 
2deb-src http://ppa.launchpad.net/ubuntugis/ppa/ubuntu lucid main

6 php apache

1sudo apt-get install apache2
2sudo apt-get install php5
3sudo apt-get install php5-dev

修改apache默认目录：
sudo vi /etc/apache2/sites-available/default

新建一个测试文件： test.php

1<?php phpinfo(); ?>

7 mongodb及其图形工具——rockmongo

初始化目录

1sudo mkdir -p /data/db/
2sudo chown `id -u` /data/db

安装rockmongo：

配置php_mongo

sudo apt-get install php-pear
sudo apt-get install libcurl3-openssl-dev
sudo pecl install mongo
修改php配置：
sudo vi /etc/php5/apache2/php.ini

在其中添加如下内容：

1extension=mongo.so
